podlogin.org: Crossclound "Pod Login" Management Tool

This site is used internally by crosscloud.js and similar libraries to help users manage their settings across various pods. It serves as the HTTP "origin" for cookies and/or local-storage across all webapps.

This site also provides a partial list of public pod providers to be used when users want a new pod.

These two functions are essentially centralized, because we know of no practical way to decentralize them with current browser technology. These are, however, only modest convenience services which could also be offered by others or integrated into browsers.
